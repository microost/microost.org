import org.microost.*;

"microost.org" is an Internet domain name for package names of open-sourced Java libraries.

When publishing a new small open-sourced Java library as a person, naming its Java package is a problem. Internet domain names are used for Java packages.

  • Everyone does not have their own personal domain name. Even if they have, personal domain names may not be the best solution as their lifetimes depend.
  • Domain names of source code hosting services (such as github.com and sf.net) are not always the best solution also due to their lifetimes.
  • Purchasing a new domain name for each library is not very reasonable.

"microost.org" intends to be a co-managed hosting of a Java package namespace in the future. It is yet owned/managed by a single person for the time being, though.

Q & A

What is "microost.org" for?

  • "microost.org" is only for hosting a Java package namespace to reserve.

What is NOT "microost.org" for?

  • "microost.org" is NOT for hosting subdomain names.
  • "microost.org" is NOT for hosting source code.
  • "microost.org" is NOT for hosting library packages.
  • "microost.org" is NOT for hosting web pages.

Is "microost.org" to be maintained farther into the future?

Not guaranteed unfortunately. The owner may retire, may be bored, or may give up. Assume "microost.org" is still just a personal domain name which is accepting requests from others. Co-owners may help the situation!

Even if the owner gives up, will try calling for successors as much as possible.

Can I get to an co-owner?

Co-owners are not publicly called for at the moment. Ping me if you and I know each other.

How is a new package name registered?

Send a pull request to the GitHub repository. See README.md there.

Are all valid registration requests unconditionally accepted?

No, at least for the time being. The owner would review pull requests.

Can I register a package name for a person or an organization, not for a project?

It is not forbidden, but not recommended.

What does "microost" mean?

"microost" is a coined word from "microroast" and "roost".

How is "microost" pronounced?

Like mike-roost.

Can I fork "microost.org" and host similar on a different domain name?

No any problems! Feel free to host your own.
